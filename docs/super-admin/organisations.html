Organisations

Partner organisations can be onboarded to Community Carpool to encourage carpooling among their employees or members. Each organisation gets a short code that can be used in referral links and tracked in analytics.

How organisations work

When a user submits a journey with an org_code, it is stored in both users.org_code and submissions.org_code. This enables:

  • Filtering submissions and matches by organisation in analytics queries
  • Organisation-specific referral links (e.g. communitycarpool.org/?org=ACME) that pre-fill the org_code on the form
  • Future: showing a user's organisation in their profile and prioritising intra-org matches

Viewing current organisations

SELECT org_id, org_code, name, domain, is_active FROM organisations ORDER BY name;

Adding a new organisation

INSERT INTO organisations (org_code, name, domain, is_active)
VALUES ('ACME', 'Acme Corporation', 'acme.com', true);

Fields:

  • org_code — short, uppercase, unique identifier used in URLs (e.g. ACME, DIFC, JVC)
  • name — full organisation name as displayed to users
  • domain — email domain (for future auto-detection of org when user signs up with a company email)
  • is_active — when false, org is excluded from dropdowns and analytics

Deactivating an organisation

UPDATE organisations SET is_active = false WHERE org_code = 'ACME';

Existing users and submissions with this org_code are not affected — the org_code remains on their records. Deactivation only prevents new submissions referencing this org.

Analytics by organisation

-- Submissions by org (top 10)
SELECT org_code, COUNT(*) as submission_count
FROM submissions
WHERE journey_status = 'active'
GROUP BY org_code
ORDER BY submission_count DESC
LIMIT 10;

-- Users by org
SELECT org_code, COUNT(*) as user_count
FROM users
WHERE org_code IS NOT NULL
GROUP BY org_code
ORDER BY user_count DESC;

Referral links

Generate org-specific links for distribution to partner organisations:

Link typeFormatEffect
Org pre-fill communitycarpool.org/?org=ORGCODE Pre-fills the organisation field on the submission form
Referral tracking communitycarpool.org/?ref=ORGCODE Tracked for analytics (future UTM support)
Combined communitycarpool.org/?org=ORGCODE&ref=ORGCODE Pre-fills and tracks

QR codes pointing to these URLs can be generated with any standard QR code tool and distributed on posters, internal communications, or emails.
