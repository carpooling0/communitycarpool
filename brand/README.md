# Community Carpool Brand Guidelines

Professional brand identity system and design standards for Community Carpool.

## 🌍 About

Community Carpool is a platform that connects neighbors to share sustainable journeys. Our brand reflects trust, optimism, and a commitment to building stronger communities through shared transportation.

## 🎨 Brand Identity

### Colors

- **Dark Forest Green** (#1B5C3A) - Trust & Stability
- **Bright Chartreuse** (#B4E035) - Energy & Growth  
- **Button Green** (#10B981) - Action & Connection
- **Text Primary** (#1F2937) - Readability

### Typography

- **Headings:** Montserrat (700, 900)
- **Body:** Inter (400, 600)
- **Code:** Source Code Pro

### Design System

- 60+ components with full specifications
- Accessibility (WCAG AA+)
- Responsive design patterns
- Mobile-first approach

## 🚀 Usage

### For Designers

1. Download design files (Figma, Sketch, Adobe XD)
2. Use the component library
3. Follow color and typography specs
4. Reference spacing and layout guidelines

### For Developers

1. Access the HTML guide for specifications
2. Use provided CSS variables
3. Reference accessibility guidelines
4. Follow responsive breakpoints

### For Marketing/Content

1. Review voice and tone guidelines
2. Check photography best practices
3. Use approved assets and logos
4. Reference brand messaging
