## 🚀 Quick Start (5 Steps)

### Step 1: Prepare Files
```bash
# Copy all files from /outputs to your local brand directory
mkdir ~/brand-guidelines
cp -r /outputs/* ~/brand-guidelines/

# Rename index file
cd ~/brand-guidelines
mv brand-site-index.html index.html
```

### Step 2: Create GitHub Repo
1. Go to https://github.com/new
2. Name: `brand`
3. Description: `Community Carpool Brand Guidelines`
4. Make it Public
5. Create

### Step 3: Push to GitHub
```bash
cd ~/brand-guidelines
git init
git add .
git commit -m "Initial commit: Brand Guidelines"
git branch -M main
git remote add origin https://github.com/communitycarpool/brand.git
git push -u origin main
```

### Step 4: Enable GitHub Pages
1. Go to: https://github.com/communitycarpool/brand/settings/pages
2. Source: `main` branch, `/root` folder
3. Click Save

### Step 5: Share the Link
1. Site goes live: https://communitycarpool.github.io/brand/
2. Add to Support panel
3. Add to Admin panel
4. Add to Super-Admin panel
5. Share with team!
